# Habit Tracker Website

## Description
This is a simple habit tracker web application built using HTML, CSS, and JavaScript.
Users can add daily habits, mark them as completed, and remove them.

## Features
- Add new habits
- Mark habits as completed
- Delete habits
- Data persistence using browser localStorage

## Concepts Demonstrated
- DOM manipulation
- Event handling
- Basic state management
- User interface design

## Technologies
- HTML
- CSS
- JavaScript

## Motivation
This project was developed as part of my independent learning process to practice
basic programming logic, user interaction, and front-end development.
